1.React nima.

React — bu foydalanuvchi interfeyslarini yaratish uchun Facebook tomonidan ishlab chiqilgan JavaScript kutubxonasi. U asosan bir sahifali ilovalar (SPA) yaratishda ishlatiladi. React yordamida dinamik va interaktiv foydalanuvchi interfeyslarini yaratish mumkin. React kompozitsion yondashuvga ega, ya'ni UI ni kichik, qayta ishlatiladigan komponentlarga ajratib yaratishga imkon beradi.


2.React’ning afzalliklari.

  • Kompontentlarga asoslangan yondashuv:UI ni kichik, oson boshqariladigan va qayta ishlatiladigan komponentlarga ajratish imkonini beradi.
  • Virtual DOM:Virtual DOM yordamida DOM manipulyatsiyasining samaradorligi oshadi, chunki faqat kerakli qismlar yangilanadi.
  • Deklarativ yondashuv: UI holatiga qarab React nima qilish kerakligini belgilashni osonlashtiradi.
  • React Native:React bilan mobil ilovalar yaratish imkonini beradi.
  • Ko‘plab kutubxonalar va ekotizimlar:React bilan ishlashda turli kutubxonalar (Redux, React Router, va boshqalar) mavjud.

3.JSX nima va qanday ishlaydi?

JSX (JavaScript XML) — bu React-da HTML va JavaScript kodlarini birlashtirgan maxsus sintaksisdir. JSX yordamida HTML kodlarini JavaScript ichida yozish mumkin. JSX aslida JavaScript kodiga transpilyatsiya qilinadi, ya'ni bu sintaksisni brauzer tushunmaydi, lekin React bu kodni React.createElement() funksiyasiga o‘zgartiradi.

4. Virtual DOM tushunchasi

Virtual DOM — bu React tomonidan yaratilgan xotirada joylashgan DOM nusxasidir. Foydalanuvchi interfeysida biror narsa o‘zgarganda, React avval Virtual DOMni yangilaydi, so‘ngra yangi Virtual DOM bilan asl DOMni taqqoslaydi va faqat o‘zgarishlarni asl DOMga kiritadi. Bu jarayon samaradorlikni oshiradi, chunki to‘liq DOMni yangilashning o‘rniga faqat zarur qismlar yangilanadi.

5. React’ni o‘rnatish (CDN va Create React App)

  • CDN orqali o‘rnatish: Agar React’ni faqat kichik loyiha uchun ishlatmoqchi bo‘lsangiz, CDN orqali o‘rnatish mumkin. Misol uchun, HTML faylga quyidagi script teglari orqali React va ReactDOM kutubxonalarini ulashing:
  • Create React App orqali o‘rnatish: Create React App — bu React ilovalarini yaratish uchun qulay boshqaruvchi vositadir. Bu vosita yordamida React ilovalarini tezda yaratish mumkin. O‘rnatish uchun terminalda quyidagilarni bajarish kerak:

    • npx create-react-app my-app

    cd my-app

    npm start

6. React komponentlari (Function va Class komponentlar)

  • Function komponentlari: Bu komponentlar faqat funksiyalar yordamida yaratiladi. Ular React’ning zamonaviy usuliga asoslangan.
  • Class komponentlari: Klassik usulda komponentlar class orqali yaratildi va render() metodidan foydalanadi.
  • Bugungi kunda Function komponentlari ko‘proq ishlatiladi, chunki ular ko‘proq optimallashtirilgan va osonroq ishlatiladi, ayniqsa React hooks bilan.

7. React’ning asosiy sintaksisi

React’ning asosiy sintaksisi quyidagi elementlardan tashkil topadi:

  • JSX: Yuqorida aytib o‘tganimizdek, JSX HTML va JavaScriptning birlashgan sintaksisi.
  • Komponentlar: Function va Class komponentlari.
  • Props: Komponentlarga ma'lumot uzatish uchun ishlatiladi. Masalan: jsx
  • State: Komponentlarning ichki holatini saqlaydi va yangilaydi. useState hooki yordamida ishlatiladi.
